# scryptor-p2p
Мессенджер P2P для конфиденциального общения с усиленной криптографией. Форк моего [другого проекта](https://github.com/clappybird01/secmess-encoder/).

📝 Немного о проекте
Проект разрабатывается энтузиастом для изучения цифровой безопасности, шифрования и защиты данных. Некоторые функции и особенности:
- **Простота установки и использования:** для работы со Scryptor нужно лишь наличие современного браузера и включенного WebRTC.
- **Конфиденциальность:** весь код, соединение и переписка выполняются локально на устройстве. Ваши данные никуда не передаются.
- **Простой интерфейс:** на выбор предоставляются разные темы, дизайн приложения не перегружен ненужными кнопками или функциями. Простая логика соединения с собеседником, возможность отправлять любые файлы.
- **Актуальные протоколы для безопасности:** строгая политика Content Security Policy (CSP), использование проверенных и современных протоколов для шифрования сообщений и вычисления общего ключа.
- **Perfect Forward Secrecy (PFS):** алгоритм Double Ratchet и протоколы с одноразовыми публичными ключами, которые вместе обеспечивают идеальную безопасность и секретность сообщений.
- **Подходит под большинство задач:** шифратор позволяет обмениваться текстом, файлами любого формата, голосовыми сообщениями без нужды в сторонних программах.
- **Пост-квантовая криптография:** Scryptor создаёт гибридные ключи с протоколами X25519 и ML-KEM, предотвращая потенциальное раскрытие переписки в будущем.

## Установка и использование
Вы всегда можете использовать самую актуальную версию Scryptor через [GitHub Pages](https://clappybird01.github.io/scryptor-p2p/) на любом устройстве с современным браузером. Также вы можете скачать все необходимые файлы из репозитория и запустить Scryptor из файла index.html. Убедитесь, что скачиваете все файлы и папки, находящиеся в репозитории, иначе приложение может работать некорректно или не работать вовсе.

На данный момент после обновления страницы переписка не сохраняется, и вам нужно заново соединяться с собеседником. В будущих обновлениях планируется реализовать сохранение переписок, контактов и множество других удобных функций для повседневного общения. Следите за обновлениями! :3
